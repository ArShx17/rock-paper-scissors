# Rock Paper Scissors – Flask Game

A simple web-based Rock Paper Scissors game built using Flask.

## Features
- User vs Computer
- Random computer choice
- Win/Lose/Tie result
- Simple and clean UI

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How To Run

1. Install Flask:
pip install flask

2. Run:
python main.py

3. Open in browser:
http://127.0.0.1:5000/
